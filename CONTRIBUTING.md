# Contributing to MiLuAssistantWeb

MiLuAssistantWeb is a CoPaw / QwenPaw-derived web edition adapted for the MiLu assistant project. This repository is primarily maintained as a personal secondary-development portfolio and as the base of the Windows desktop package.

If you want to compare this repository with the upstream project, focus on the MiLu namespace changes, console branding, local model/provider defaults, runtime directory isolation, and public cleanup work.

For issues or suggestions, please open a GitHub issue in this repository.
